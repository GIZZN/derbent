# Функциональность изменения статуса с DRAFT на ACTIVE

## ✅ Уже реализовано

Функциональность для изменения состояния вакансий с DRAFT на ACTIVE с помощью кнопки уже полностью реализована в разделе "Мои вакансии".

## Как это работает

### 1. **Отображение статуса**
- В карточке вакансии отображается текущий статус (DRAFT, ACTIVE, PAUSED, CLOSED)
- Статус показан с цветовой индикацией:
  - 🔘 **DRAFT** (Черновик) - серый цвет
  - 🟢 **ACTIVE** (Активна) - зеленый цвет
  - 🟡 **PAUSED** (Приостановлена) - желтый цвет
  - 🔴 **CLOSED** (Закрыта) - красный цвет

### 2. **Кнопка "Опубликовать"**
- Кнопка появляется **только** для вакансий со статусом DRAFT
- Расположена в правом верхнем углу карточки вакансии
- Имеет зеленый цвет и иконку стрелки вверх

### 3. **Процесс публикации**
- При нажатии на кнопку "Опубликовать":
  1. Статус вакансии меняется с DRAFT на ACTIVE
  2. Кнопка показывает состояние загрузки (спиннер)
  3. Текст меняется на "Публикация..."
  4. После успешной публикации список автоматически обновляется
  5. Кнопка исчезает (так как статус больше не DRAFT)

### 4. **Визуальная обратная связь**
- **Hover-эффект**: кнопка поднимается и появляется тень
- **Состояние загрузки**: анимированный спиннер
- **Автообновление**: список обновляется без перезагрузки страницы

## Техническая реализация

### API Endpoint:
```typescript
POST /jobs/{jobId}/publish
```

### Компонент:
```jsx
{job.status === 'DRAFT' && (
  <button 
    onClick={() => handlePublishJob(job.id)}
    disabled={isPublishing}
    className={styles.publishButton}
    title="Опубликовать вакансию"
  >
    {isPublishing ? 'Публикация...' : 'Опубликовать'}
  </button>
)}
```

### Обработчик:
```typescript
const handlePublishJob = async (jobId: string) => {
  try {
    await publishJob(jobId).unwrap();
    refetch(); // Обновление списка
  } catch (error) {
    console.error('Failed to publish job:', error);
  }
};
```

## Пользовательский опыт

### Для HR:
1. **Создает вакансию** → статус DRAFT
2. **Видит кнопку "Опубликовать"** в карточке вакансии
3. **Нажимает кнопку** → статус меняется на ACTIVE
4. **Кнопка исчезает** → вакансия опубликована
5. **Может редактировать** опубликованную вакансию

### Визуальные индикаторы:
- ✅ **Статус вакансии** отображается в правом верхнем углу
- ✅ **Кнопка публикации** только для черновиков
- ✅ **Цветовая индикация** для разных статусов
- ✅ **Анимация загрузки** при публикации
