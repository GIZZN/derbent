# 📄 Руководство по системе структурированных резюме

## 🎯 Обзор

Система резюме была полностью переработана для поддержки структурированных данных. Теперь каждое резюме - это объект с полями опыта работы, навыков, образования и других данных.

## ✨ Основные возможности

- **Множественные резюме** - пользователь может создать несколько резюме
- **Структурированные данные** - опыт, навыки, образование хранятся в JSON
- **Основное резюме** - одно резюме может быть помечено как основное
- **Публичность** - резюме может быть публичным или приватным
- **Дублирование** - возможность копировать существующие резюме
- **Детальный просмотр** - красивое отображение всех данных резюме
- **Редактирование** - полное редактирование всех секций резюме

## 🏗️ Структура данных

### Основные секции резюме:

1. **Основная информация**
   - Название резюме
   - Краткое описание
   - Цель/цели
   - Статус (основное/публичное)

2. **Навыки**
   - Название навыка
   - Уровень (1-5)
   - Категория

3. **Опыт работы**
   - Должность и компания
   - Период работы
   - Описание обязанностей
   - Достижения
   - Используемые технологии

4. **Образование**
   - Учебное заведение
   - Степень и специальность
   - Период обучения
   - Средний балл (GPA)
   - Описание

5. **Проекты**
   - Название проекта
   - Описание
   - Период работы
   - Технологии
   - Ссылки (сайт, GitHub)

6. **Достижения**
   - Название достижения
   - Описание
   - Дата
   - Категория

7. **Языки**
   - Название языка
   - Уровень владения
   - Сертификаты

8. **Сертификаты**
   - Название сертификата
   - Организация-выдаватель
   - Дата получения
   - ID сертификата
   - Ссылка на сертификат

## 🚀 Как использовать

### Создание резюме

1. Нажмите кнопку "Создать резюме"
2. Введите название резюме
3. Нажмите "Создать"

### Просмотр резюме

1. Нажмите кнопку "👁" (глаз) рядом с резюме
2. Откроется детальный просмотр со всей информацией
3. В режиме просмотра доступны действия:
   - Редактировать
   - Дублировать
   - Сделать основным (если не основное)
   - Изменить публичность

### Редактирование резюме

1. Нажмите кнопку "✏️" (карандаш) рядом с резюме
2. Откроется редактор с навигацией по секциям
3. Переключайтесь между секциями с помощью вкладок
4. Заполняйте необходимые поля
5. Нажмите "Сохранить изменения"

### Управление резюме

- **Просмотр**: Кнопка с глазом - открывает детальный просмотр
- **Редактирование**: Кнопка с карандашом - открывает редактор
- **Дублирование**: Кнопка с копией - создает копию резюме
- **Удаление**: Кнопка с корзиной - удаляет резюме

### Фильтрация и поиск

- **Поиск**: Введите название резюме в поле поиска
- **Фильтры**: 
  - Основное резюме (Да/Нет/Все)
  - Публичность (Публичные/Приватные/Все)
  - Сортировка (по дате создания/обновления/названию)
  - Порядок (по возрастанию/убыванию)

## 🔧 Технические детали

### API Endpoints

- `POST /resumes` - создание резюме
- `GET /resumes` - получение списка резюме
- `GET /resumes/{id}` - получение резюме по ID
- `PUT /resumes/{id}` - обновление резюме
- `DELETE /resumes/{id}` - удаление резюме
- `POST /resumes/{id}/set-default` - установка основного резюме
- `POST /resumes/{id}/duplicate` - дублирование резюме

### Компоненты

- `ResumeViewer` - компонент для отображения резюме
- `ResumeEditor` - компонент для редактирования резюме
- `ConfirmationDialog` - диалоги подтверждения
