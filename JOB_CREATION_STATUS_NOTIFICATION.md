# Уведомление о статусе созданной вакансии

## Описание изменений

Добавлено красивое уведомление с отображением статуса вакансии после её создания. Пользователь видит название созданной вакансии и её текущий статус (ACTIVE, PAUSED, CLOSED).

## Изменения в файлах

### 1. `src/app/jobs/create/page.tsx`
- **Добавлено состояние**: `showSuccessMessage` и `createdJobInfo` для управления уведомлением
- **Обновлена функция handleSubmit**: сохранение информации о созданной вакансии
- **Добавлена функция getStatusLabel**: перевод статусов на русский язык
- **Добавлен компонент уведомления**: модальное окно с информацией о статусе
- **Автоматический редирект**: через 3 секунды после создания

### 2. `src/app/jobs/create/create.module.css`
- **Добавлены стили уведомления**: `.successOverlay`, `.successMessage`
- **Анимации**: плавное появление и исчезновение
- **Стили статусов**: разные цвета для ACTIVE, PAUSED, CLOSED
- **Адаптивный дизайн**: корректное отображение на мобильных устройствах

## Функциональность

### Уведомление о создании:
- ✅ **Красивое модальное окно** с анимацией появления
- ✅ **Название вакансии** в кавычках
- ✅ **Статус с цветовой индикацией**:
  - 🟢 ACTIVE (Активна) - зеленый цвет
  - 🟡 PAUSED (Приостановлена) - желтый цвет

- ✅ **Автоматический редирект** через 3 секунды
- ✅ **Адаптивный дизайн** для всех устройств

### Анимации:
- **Fade In**: плавное появление overlay
- **Slide In**: анимация появления модального окна

### Логика отображения статуса:
```typescript
const getStatusLabel = (status: string) => {
  switch (status) {
    case 'ACTIVE': return 'Активна';
    case 'PAUSED': return 'Приостановлена';
    case 'CLOSED': return 'Закрыта';
    default: return status;
  }
};
```
